### 基本信息

AutoServer是一款跨平台的安卓自动化框架,在android adb,uiautomator,appium的基础上加入了http协议,你可以非常方便的编程你的自动化操作，它类似于按键精灵,触动精灵,Auto.js，支持Python,Lua,java,javascript等语言,实现真正的跨平台开发。
